ShopFire Firestore Setup

Note: This page helps you set up your Firestore database with initial data. Make sure you have already created your Firestore database in the Firebase Console.

Prerequisites

  1. Create your Firestore Database in the Firebase Console
  2. Enable Firebase Authentication with Email/Password provider
